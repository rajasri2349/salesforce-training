# What is LWC?

LWC (Lightning Web Components) is Salesforce’s modern UI framework used to build fast and reusable user interfaces.

LWC uses:
- HTML
- JavaScript
- CSS
- Metadata XML

It helps developers create modern and efficient Salesforce applications.

---

# Why Salesforce Uses LWC

Salesforce uses LWC because:
- Better performance
- Reusable components
- Faster UI rendering
- Modern web standards
- Easier maintenance
- Better user experience

LWC follows component-based architecture.

# Why Reusable Components Are Useful

Reusable components help:
- Reduce duplicate code
- Improve maintenance
- Increase development speed
- Create consistent UI
- Simplify updates

One component can be used in multiple places.

---

# Frontend vs Backend Logic

| Frontend / UI | Backend / Apex |
|---------------|----------------|
| Button clicks | Data validation |
| Form display | Fee calculation |
| Notifications display | Database updates |
| User interaction | Business logic |
| UI styling | Trigger execution |

---

# Examples

## Button Click
Belongs to:
- Frontend/UI

Reason:
- User interacts directly with interface.

---

## Data Validation
Belongs to:
- Backend/Apex

Reason:
- Business rules must be securely enforced.

---

## Fee Calculation
Belongs to:
- Backend/Apex

Reason:
- Important business calculations should run securely.

---

## Notification Display
Belongs to:
- Frontend/UI

Reason:
- Notifications are shown to users on screen.

# Reflection

Modern enterprise systems use component-based UI architecture because it improves:
- Reusability
- Performance
- Scalability
- Maintenance
- Team collaboration

Component-based systems are easier to manage and update.

# Revision Questions

## 1. What is a component?
A component is a reusable part of a user interface.

---

## 2. Why are reusable components useful?
They reduce duplicate work and improve maintenance.

---

## 3. Difference between frontend and backend?

Frontend:
- User interface and interaction

Backend:
- Business logic and database operations

---

## 4. Why did Salesforce move toward LWC?
To improve performance and adopt modern web standards.

---

## 5. Why is UI important in enterprise systems?
Good UI improves productivity and user experience.

---

## 6. Why should systems separate UI and business logic?
It improves security and maintainability.

---

## 7. What security risks exist in enterprise applications?
Unauthorized access and data leakage.

## 8. Why should developers think modularly?
Modular systems are easier to build, test and maintain.
