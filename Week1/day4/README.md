## Data and Actions in Flows

### What are Data and Actions in Salesforce Flows?

Data and Actions in Flows help automate business processes by working with Salesforce records automatically.

### Key Concepts Learned
- Creating and updating records
- Retrieving data from Salesforce objects
- Using decisions and conditions
- Automating record updates
- Flow logic and automation steps

### Why It Matters
Data and Actions in Flows reduce manual work, improve accuracy, and help businesses automate repetitive tasks without coding.
